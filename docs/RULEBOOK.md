# Swipe Breakout 룰북

게임 규칙·스테이지 진행·특수 조건을 정리한 문서다. 개별 아이템의 효과·연출·제작 상태는 [ITEM_CHECKLIST.md](ITEM_CHECKLIST.md)에서 관리한다.

## 목표와 기본 진행

| 단계 | 규칙 |
| --- | --- |
| 조준 | 발사 위치에서 위쪽으로 드래그해 방향을 정한다. |
| 발사 | 보유 공 전체가 같은 방향으로 발사된다. |
| 공격 | 공이 블록에 닿으면 기본 피해 1을 준다. 강화볼은 해당 배율만큼 피해가 증가한다. |
| 턴 종료 | 모든 공이 바닥으로 돌아오면 발사 위치가 첫 착지 지점으로 이동하고, 블록·필드 아이템이 한 줄 내려온다. |
| 클리어 | 강철을 제외한 모든 블록을 제거하면 스테이지 결과와 궁극기 보상을 확인한다. |
| 게임오버 | 파괴 가능한 블록이 위험선 아래로 내려오면 종료한다. 보호막은 이 상황을 한 번 되돌린다. |

## 보드와 스테이지

| 항목 | 규칙 |
| --- | --- |
| 보드 | 가로 8열. 파괴 가능한 블록의 위험선은 10번째 행 아래다. |
| 스테이지 1~5 | 수동으로 만든 고정 레이아웃을 사용한다. |
| 스테이지 6+ | 스테이지 번호를 시드로 쓰는 절차적 레이아웃을 사용한다. 같은 스테이지는 항상 같은 배치다. |
| 블록 체력 | 절차적 스테이지는 스테이지가 오를수록 기본 체력이 증가하며 최대 50이다. |
| 특수 블록 | 절차적 스테이지의 레이저 블록은 6~15에서 1개, 16~30에서 2개, 31+에서 3개다. 6행 이상 생성되는 스테이지는 최하단 생성 행에 레이저 1개를 보장한다. 강철 블록은 1개 배치된다. |
| 필드 아이템 | 모든 스테이지에 보호막을 배치한다. 1~15는 최대 4개, 16~30은 최대 5개, 31+는 최대 7개를 배치하며 후반으로 갈수록 강화볼 ×3·×4와 함정 비중이 늘어난다. |

## 공 수와 속도

| 스테이지 | 보유 공 상한 |
| --- | ---: |
| 1~20 | 30 |
| 21~30 | 40 |
| 31~40 | 50 |
| 41~50 | 60 |
| 51~70 | 70 |
| 71+ | 80 |

공이 30개를 넘으면 발사 속도가 공 수에 비례해 최대 약 1.36배까지 증가한다. 화면이 공으로 가득 찬 뒤 게임이 늘어지는 상황을 줄이기 위한 규칙이다.

## 특수 조건

| 조건 | 규칙 |
| --- | --- |
| 레이저 블록 | 파괴되면 같은 행의 파괴 가능한 블록을 모두 제거한다. |
| 강철 블록 | 모든 공격에 면역이며 공만 반사한다. |
| 블랙홀 필드 아이템 | 스테이지 10부터 3스테이지 간격으로 등장한다. 주변 공의 궤도를 휘게 하고 흡수한다. 5초마다 안전한 하단 빈칸으로 이동한다. 흡수 횟수는 1~30스테이지에서 1회, 31+에서 2회다. |
| 공 추가 | 획득 즉시 충돌한 공의 꼬리에 새 공이 합류하며, 다음 발사 공 수도 1개 증가한다. |
| 공 감소 함정 | 공을 1개 줄이되 최소 1개는 유지한다. |

## 스테이지 평가와 궁극기 보상

| 별 | 획득 조건 |
| --- | --- |
| 기본 1개 | 스테이지 클리어 |
| +1개 | 스테이지 목표 점수 달성 |
| +1개 | 목표 시간 이내 클리어 |
| +1개 | 목표 시간의 80% 이내 클리어 |
| +1개 | 목표 시간의 60% 이내 클리어 |

궁극기를 사용하면 사용 횟수만큼 별이 줄어들며, 최종 별은 최소 1개다. 목표 점수와 시간은 스테이지의 블록 체력·개수·강철·아이템 구성으로 계산된다.

| 최종 별 | 궁극기 보상 확률 |
| --- | ---: |
| 1개 | 0% |
| 2개 | 20% |
| 3개 | 50% |
| 4개 | 80% |
| 5개 | 100% |

보상은 선택지가 아닌 가중치 랜덤 1개다. 성장형인 연쇄 번개·핵융합 연쇄·미사일 포화와 운석 충돌은 1스테이지부터 낮은 레벨로 등장한다. 고정형 블랙홀 붕괴·십자포화는 11스테이지, 반물질 폭탄·궤도 레이저는 21스테이지부터 후보에 추가된다.

## 보관형 궁극기

| 항목 | 규칙 |
| --- | --- |
| 보관 | 하단 슬롯 2개에 저장한다. 슬롯이 가득 찬 상태에서 보상을 받으면 교체할 슬롯을 선택하거나 보상을 건너뛴다. |
| 사용 | 발사 대기 상태에서 슬롯을 누르고 게임판 위치를 선택한다. 사용해도 턴이 지나가거나 블록이 내려오지 않는다. |
| 성장 레벨 | 연쇄 번개·핵융합 연쇄·미사일 포화는 보상으로 획득한 레벨에 따라 대상 수가 달라진다. 상세 수치는 [아이템 체크리스트](ITEM_CHECKLIST.md#궁극기-성장-레벨)를 따른다. |

## 기록과 랭킹

게임오버 또는 챌린지 포기 후 기록을 등록할 수 있다. 순위는 점수 내림차순, 도달 스테이지 내림차순, 순수 플레이 시간 오름차순으로 결정한다. 순수 플레이 시간에는 조준·공 비행만 포함하며, 대기·일시정지 시간은 포함하지 않는다.
