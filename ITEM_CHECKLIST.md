# 아이템 제작 체크리스트

아이템을 추가하거나 효과·수치·연출을 수정할 때 코드와 이 문서를 같은 커밋에서 함께 갱신한다. 새 아이템은 구현 전에 미완료 항목으로 추가하고, 게임 규칙·화면·검증이 끝난 뒤 `[x]`로 변경한다.

## 화면 배치 아이템

- [x] 공 추가 (`multiball`) — 획득 즉시 다음 발사 공을 1개 추가한다. 최대 30개.
- [x] 폭탄 (`bomb`) — 주변 3×3 범위의 파괴 가능한 블록에 피해 1을 준다.
- [x] 보호막 (`shield`) — 블록 하강으로 게임오버될 때 한 번 되돌린다.
- [x] 강화볼 ×2 (`power`) — 2회 발사 동안 공의 피해를 2배로 만든다.
- [x] 강화볼 ×3 (`power3`) — 2회 발사 동안 공의 피해를 3배로 만든다.
- [x] 강화볼 ×4 (`power4`) — 2회 발사 동안 공의 피해를 4배로 만든다.
- [x] 공 감소 함정 (`trap`) — 공을 1개 줄인다. 최소 1개는 유지한다.
- [x] 블랙홀 (`blackhole`) — 주변 공의 궤도를 휘게 하고 충전 수만큼 흡수하며 5초마다 안전한 빈칸으로 이동한다.

## 특수 블록

- [x] 레이저 블록 (`laser`) — 파괴되면 같은 가로줄의 파괴 가능한 블록을 모두 제거한다.
- [x] 강철 블록 (`steel`) — 모든 공격에 파괴되지 않고 공만 반사한다.

## 보관형 궁극기

궁극기는 발사 대기 상태에서 슬롯을 누른 뒤 게임판 위치를 선택해 사용한다. 유효한 대상이 없으면 소비하지 않으며 강철 블록은 모든 궁극기에 면역이다.
기본 연출은 PixiJS의 도형·선·블러·투명도로 구현하고, 효과만으로 식별하기 어려울 때 아이콘 외 추가 이미지 리소스를 제작한다.
효과음은 기존 사운드를 재사용한다. 레이저 계열은 `laser`, 그 외 대부분의 대규모 파괴 궁극기는 `bomb`을 사용한다.

- [x] 반물질 폭탄 (`antimatter`) — 선택 지점을 중심으로 5×5 범위의 파괴 가능한 블록을 체력과 무관하게 제거한다. 보라색 충격파와 화면 섬광, 폭탄 효과음으로 연출하며 `public/images/ultimates/antimatter.png` 아이콘을 사용한다.
- [x] 궤도 레이저 (`orbitalLaser`) — 선택 열을 중심으로 세로 3열의 파괴 가능한 블록을 체력과 무관하게 제거한다. 전체 높이 광선과 레이저 효과음으로 연출하며 `public/images/ultimates/orbital-laser.png` 아이콘을 사용한다.
- [x] 연쇄 번개 (`chainLightning`) — 선택 지점에서 가까운 파괴 가능한 블록 최대 12개를 체력과 무관하게 제거한다. 목표 사이를 잇는 전기선과 폭탄 효과음으로 연출하며 `public/images/ultimates/chain-lightning.png` 아이콘을 사용한다.
- [ ] 운석 충돌 — 선택 지점 중심부의 블록을 즉시 제거하고 외곽 블록에 피해를 준다. 빛나는 원과 꼬리, 충돌 충격파, 폭탄 효과음으로 연출한다.
- [ ] 블랙홀 붕괴 — 선택 범위의 블록을 한 점으로 끌어당긴 뒤 폭발시켜 제거한다. 기존 블랙홀의 원·타원 효과와 폭탄 효과음을 재사용한다.
- [ ] 십자포화 — 선택 위치를 중심으로 가로 3줄과 세로 3줄의 블록을 제거한다. 교차하는 다중 레이저와 레이저 효과음으로 연출한다.
- [ ] 핵융합 연쇄 — 선택 블록에서 시작해 주변 블록으로 폭발을 전염시킨다. 블록별 순차 점멸과 확장 원, 폭탄 효과음으로 연출한다.
- [ ] 미사일 포화 — 다수의 파괴 가능한 블록을 자동 조준해 동시에 공격한다. 삼각형 탄두와 꼬리 선, 폭탄 효과음으로 연출한다.
